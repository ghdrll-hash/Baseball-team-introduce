# 2025-2026 KBO 구단 소개 웹사이트
삼성 라이온즈, 한화 이글스, LG 트윈스의 2025시즌 리뷰와 2026시즌 전망을 소개하는 반응형 웹사이트 프로젝트입니다.

## 프로젝트 소개
이 프로젝트는 KBO 리그의 인기 구단들의 역사, 시즌 성적, 대표 선수, 그리고 응원가 정보를 한눈에 볼 수 있도록 제작되었습니다.
각 구단의 상징색(파랑, 주황, 빨강)을 테마로 디자인하였으며, Flexbox를 활용하여 모바일과 PC 모두에서 최적화된 화면을 제공합니다.

## 주요 기능
* 메인 화면 (Split Landing Page): 화면을 3등분 하여 각 구단으로 연결되는 직관적인 내비게이션을 제공합니다. 마우스 오버 시 영역이 확장되는 인터랙티브 효과를 적용했습니다.
* 구단별 상세 페이지:
    * 삼성 라이온즈: 2025 정규시즌 4위 및 한국시리즈 진출 성과, 원태인/디아즈 선수 소개.
    * 한화 이글스: 류현진 복귀 효과와 2025 포스트시즌 진출, 강백호 영입 및 다이너마이트 타선 전망.
    * LG 트윈스: 2025 통합 우승 달성 및 2년 연속 우승 도전, 임찬규/박해민 선수 활약상.
* 미디어 연동: 각 구단의 대표 응원가 유튜브 링크를 버튼 형태로 제공합니다.
* 반응형 디자인: 데스크톱에서는 가로 배치, 모바일 환경에서는 세로 배치로 자동 전환됩니다.

## 사용 기술
* HTML5: 시맨틱 태그를 활용한 문서 구조화
* CSS3: Flexbox 레이아웃, CSS Transitions(애니메이션), Media Queries(반응형), CSS Variables(테마 컬러)
